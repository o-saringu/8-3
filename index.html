プログラム復習コンテンツ

図形の移動

サンプルコードの結果

サンプルコード

　　 int radius=40;

　　 float x=110;

　　 float speed=0.5;

　　 int direction=1;

　　 　

　　 void setup(){

　　 　size(240, 120);

　　 　ellipseMode(RADIUS);

　　 }

　　 void draw(){

　　 　background(0);

　　 　x+=speed*direction;

　　 　if((x>width-radius)||(x　　 　　direction=-direction;　　 //方向転換

　　 　}

　　 　if(direction==1){

　　 　　arc(x, 60, radius, radius, 0.52, 5.76); //右向き

　　 　}else{

　　 　　arc(x, 60, radius, radius, 3.67, 8.9); //左向き

　　 　}

　　 }

説明文

　流れるような動きを表現するためにはfloat型の変数を用いる。

　draw()が実行されるたびに図形が移動し動いているように見える。

　例では、図形が壁に当たると方向転換して逆向きに移動するプログラムである。

　図形が端に達したらdirection変数の符号を反転させ、if文で判定を行う。
